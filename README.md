# nikgapps.github.io
NiKGapps Website

To setup, install [Ruby v3.4.8](https://github.com/oneclick/rubyinstaller2/releases/download/RubyInstaller-3.4.8-1/rubyinstaller-devkit-3.4.8-1-x64.exe)

Then execute following command in command prompt to install jekyll and bundler

```bash
gem install jekyll bundler
```

navigate to working directory and execute 

```bash
bundle install
```

to run the project execute the following

```bash
bundle exec jekyll serve
```
